 E-Commerce Website

Nostra is a simple front-end e-commerce website built using HTML, CSS, and JavaScript.
This project is created to practice DOM manipulation, search functionality, product filtering, and responsive design.

It is a beginner-friendly project for learning how an online shopping website works.

🚀 Features

🏠 Home page with hero section

🛒 Product listing section

🔍 Product search functionality

🏷️ Category filtering

🧾 Product cards with image, name, and price

📱 Fully responsive design

🎨 Clean and simple UI

🛠️ Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and layout

JavaScript  – Interactivity and DOM manipulation
